# UFTEnLinea

# ¿Que es?
No es mas que un simple sistema que te muestra en pantalla y te notifica de las tareas que tienes pendientes de entregar en los proximos dias.
# ¿Como funciona?
Simple, la aplicacion necesita tu usuario y contraseña de la plataforma virutal de la Universidad Fermin Toro, con estos datos se envia una peticion web a un servidor propio y retorna los datos a mostrar
# ¿Como obtenemos los datos?
Los datos son extraidos directamente del sistema de dicha universidad.
